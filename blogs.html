Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element in the DOM are- document.getElementById("ID-name"), document.getElementsByClassName("Class-name"), document.getElementsByTagName("p"), document.querySelector(".Class-name"), document.querySelectorAll(".Class-name")

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML allows reading and modifying HTML content, meaning it processes and renders any HTML tags inside an element. innerText retrieves or sets only the visible text of an element, ignoring any hidden content or HTML tags. textContent is similar to innerText but captures all text within an element, including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where a parent element is used to handle events for its child elements.Instead of adding event listeners to multiple child elements, attach a single event listener to their common parent.This works because of event bubbling, where an event moves up from the target element to its ancestors.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in JavaScript where an event starts from the target element and moves up through its parent elements. This means when an event (like a click) happens on a child element, it first runs its event handler, then triggers event handlers on its parent, grandparent, and so on, until it reaches the root
